# vetka-elisya-runtime

Elisya orchestration runtime for model routing, provider registry, and execution.

## Status
- Source of truth: monorepo danilagoleen/vetka
- Mirror sync: automated via subtree publish workflow
- Stability: experimental / fast-moving

## What Is Included
This repository contains the src/elisya module extracted from VETKA.
It is intended for modular open-source collaboration and targeted contributions.

## Stack
- Python
- Multi-provider model routing
- Runtime fallback and key roaming

## Development
1. Fork this repository.
2. Create a branch: feature/short-name or fix/short-name.
3. Use Conventional Commits (for example, feat(api): add health endpoint).
4. Open a Pull Request with context and tests.

## Release Policy
- Semantic versioning is used (vMAJOR.MINOR.PATCH).
- Changelog is maintained in CHANGELOG.md.
