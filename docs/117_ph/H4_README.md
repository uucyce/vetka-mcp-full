## Critical Code Sections

### Current Problem (No Error Handling)
```python
def _save_tasks(self, tasks: Dict[str, Any]):
    TASKS_FILE.write_text(json.dumps(tasks, indent=2, default=str))
    # ^ Fails silently if read-only
```

### Proposed Solution (Phase 117.3)
```python
def _update_task(self, task: PipelineTask):
    # Try 1: Primary
    if self._save_tasks(tasks):
        return
    # Try 2: Qdrant (recommended)
    if self._upsert_task_to_qdrant(task):
        return
    # Try 3: TMPDIR
    if self._save_to_tmpdir(task):
        return
    # Try 4: Emergency in-memory
    self._save_to_emergency_cache(task)
```
