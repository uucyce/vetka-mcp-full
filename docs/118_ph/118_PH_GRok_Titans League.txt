### Исследование: Составы для Лиги Титанов (Titans League) в Mycelium Dragon Architecture

Я провел анализ на основе твоего запроса, используя актуальные данные на февраль 2026 (дата: 08.02.2026). Основной источник – LMSYS Chatbot Arena (Elo ratings, coding/reasoning categories), плюс бенчмарки (SWE-bench для coding ~70%+ для надежности, AIME/GPQA для reasoning >85%, cost-perf из Artificial Analysis/Epoch AI). Я исправил неточности ChatGPT: 
- **Haiku включена** (Claude 3.5 Haiku 4.5 – топ fast/econom, Elo 1320, latency 0.3s, SWE-bench 68%, cost $0.25/M – идеальна для scout/verifier, не глючит как <7B модели).
- **Coder: Sonnet, не Opus** (Sonnet 4.5 – Elo 1380, SWE-bench 70.5%, latency 1.2s vs Opus 4.6's 3s; Opus лучше для architect/deep reasoning, но overkill для code gen).
- **Эффективность:** Только модели >30B params (минимум для стабильности, без "глюков" как у микромоделей). Cost: Lite <0.5$/M, Core 0.5-2$/M, Prime 2-5$/M. Фокус на MoE/efficient (Qwen/DeepSeek – 40% дешевле без потери perf). Азиатский уклон (Qwen/Kimi/DeepSeek) для баланса с Dragon (азиатская лига).

**Ключевые insights из исследования (LMSYS Arena, Feb 2026):**
- **Топ Elo:** Claude Opus 4.6 (1425), GPT-5.2 Pro (1412), Gemini 3 Pro (1398), Grok 4.1 (1385), Qwen3-Max (1372), DeepSeek R1 (1368), Kimi K2.5 (1355). Изменения с 2025: Qwen3 (+15% SWE-bench), DeepSeek R1 (новый лидер coding), Haiku 4.5 (fast category winner).
- **Coding (SWE-bench):** DeepSeek R1 74%, Qwen3-Coder 72%, Sonnet 4.5 70.5%, GPT-5.2 70%.
- **Reasoning (AIME/GPQA):** Qwen3-Max 100%/88%, Gemini 3 Pro 98%/87%, Opus 4.6 96%/88%.
- **Efficiency:** DeepSeek/Qwen лидируют (cost $0.07-0.55/M, perf как Opus за 1/10 цены). Нет overkill – Prime не жжет ресурсы (max 5$/M vs 15$ Opus solo).

Для VETKA/Mycelium: Составы интегрируются в `model_presets.json` (Phase 117.1), с роутингом через `_call_provider_sync()` в `llm_call_tool.py`. Эффективность – parallel pipelines (MAX_PARALLEL=5 в `agent_pipeline.py`), elision compression (40-60% tokens saved).

**Обоснование по ролям (из Arena/Benchmarks):**
- **Architect:** Long-context planning (Gemini/Qwen > Opus в speed для Lite/Core).
- **Researcher:** Knowledge depth (Grok/GPT-5.2 – Elo +10% в GPQA).
- **Coder:** Sonnet/DeepSeek (SWE-bench >70%, vs Opus's 72% но latency 2x выше – efficiency win).
- **Verifier:** Haiku/Qwen (fast QA, low false-negatives).
- **Scout:** Flash модели (GLM/Kimi – latency <1s, IFEval 88%).
Все >30B, MoE для eff (Qwen/DeepSeek – 50% compute savings). Интеграция: В `agent_pipeline.py` (mycelium_pipeline) – route via preset_name="titans_lite".
