## 🚨 ГЛАВНАЯ ПРОБЛЕМА

**Ответы обрезались на ~7000 токенов потому что:**

1. **OpenRouter API вызов** был ограничен **2000 токенами**
2. **Anthropic API вызов** был ограничен **4096 токенами** 
3. **Все агенты** имели лимиты **800-1000 токенов**
4. **Response formatter** уже был исправлен (лимиты убраны)

```
ПОТОК: UI → response_formatter (✅) → api_gateway (❌ 2000/4096) → провайдер
РЕЗУЛЬТАТ: Обрезка на 2000-4096 токенах независимо от UI
```
