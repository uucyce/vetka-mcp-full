## What is VETKA?

**V**isual **E**nhanced **T**ree **K**nowledge **A**rchitecture

Where **E** stands for:
- **Environment** — 3D spatial workspace
- **Engagement** — Multi-model AI collaboration
- **Education** — Learning from every interaction
- **Ecology** — Sustainable knowledge growth
- **Elisium & Elisya** — Compression and contextual memory systems

VETKA is a 3D spatial knowledge management system that unifies:
- **Multi-model AI orchestration** (Grok, GPT, Claude, Gemini, Ollama)
- **Semantic search** via Qdrant vector database
- **MCP integration** for Claude Code and external tools
- **Real-time file watching** with automatic indexing
- **Group chat** with AI agents and role-based workflows

### Key Insight
VETKA is building itself. 95% of research by Grok, verified by GPT, implemented by Claude.

## Key Features

### Model Duplication (Phase 94.4)
Same model appears twice if available from multiple sources:
- `Grok 4 (xAI)` — Direct API
- `Grok 4 (OR)` — via OpenRouter

### Routing Fix (Phase 94.8)
`x-ai/grok-4` correctly routes to OpenRouter, not direct xAI API.

### Group Chat with Roles
4 roles: PM, Architect, Worker, Observer
Smart Reply Decay prevents infinite loops.

### MCP Tools (18 registered)
Semantic search, file operations, git, camera focus, and more.
