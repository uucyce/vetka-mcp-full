### Лучшие open-source решения и библиотеки (2026)

#### 1. **Движок и оболочка браузера**
- **Tauri WebView** (твой текущий) — уже используется в `commands.rs` (`open_research_browser`, `open_external_webview`).
- **tauri-plugin-browser** (community, 2025–2026) — готовый wrapper для полноценного браузера.
- **Comet Browser** (open-source Tauri) — лучший референс по UI (vertical tabs, workspaces).
- **Arc Browser fork** (есть несколько open-source клоны на Tauri).
- **SigmaOS** — Tauri + React, очень близко к твоему стилю.

**Рекомендация**: Не тяни Chromium (тяжёлый). Оставайся на **native WebView** + кастомный UI в React.

#### 2. **Импорт из других браузеров**
**Rust (самый надёжный путь)**:
- `chrome-profile` — парсит Bookmarks, History, Cookies, Passwords.
- `rusqlite` + `places.sqlite` (Firefox).
- `keyring` — читает пароли из Keychain / Credential Manager.
- `cookie-store` + `chrome-cookie` — cookies с decryption.

**JS/TS helpers**:
- `browser-pass` (для паролей).
- `chrome-bookmarks-parser`.
- `firefox-profile` (Node).

**Готовые инструменты**:
- **Browser Profile Importer** (open-source CLI) — можно встроить.
- **Bitwarden CLI** — для импорта паролей + 2FA.

#### 3. **Аутентификация и биометрия**
- **macOS**: `LocalAuthentication` (Touch ID / Face ID) — через Rust FFI или tauri-plugin.
- **Windows**: Windows Hello — `tauri-plugin-biometric`.
- **Linux**: `fprintd` или `pam`.
- **2FA**: `totp-rs` (Rust) + импорт QR-кодов из Authy/Google.
- **Passkeys / WebAuthn**: Встроено в WebView.

**Лучший плагин**: `tauri-plugin-biometric` (2026 версия) — поддерживает все платформы.

#### 4. **Сохранение всего в VETKA**
- Закладки → **узлы в knowledge graph** (с тегами, кластерами).
- Cookies / сессии → **encrypted blobs** в Qdrant.
- Пароли → **keyring + VETKA vault**.
- Расширения → **vetka_extensions/** папка + manifest.

**Преимущество**: При переносе в новый компьютер — один клик "Export VETKA Profile" → всё (закладки, логины, 2FA, workspaces) переносится.

// MARKER_148.BROWSER_IMPORT_FULL — Полный импорт профилей браузеров в VETKA
// Поддержка: Chrome, Firefox, Arc, Opera, Edge, Brave
// Импортирует: закладки, cookies, пароли, 2FA (TOTP), расширения, историю

use serde::{Deserialize, Serialize};
use std::path::Path;
use tauri::{AppHandle, Runtime};

#[derive(Debug, Serialize, Deserialize)]
pub struct ImportResult {
    pub success: bool,
    pub browser: String,
    pub bookmarks_imported: u32,
    pub passwords_imported: u32,
    pub cookies_imported: u32,
    pub totp_imported: u32,
    pub errors: Vec<String>,
}

#[derive(Debug, Serialize, Deserialize)]
pub struct BrowserProfile {
    pub browser: String,
    pub profile_path: String,
    pub import_bookmarks: bool,
    pub import_passwords: bool,
    pub import_cookies: bool,
    pub import_2fa: bool,

// =============================================
// Главная команда
// =============================================
#[tauri::command]
pub async fn import_browser_profile<R: Runtime>(
    app: AppHandle<R>,
    profile: BrowserProfile,
) -> Result<ImportResult, String> {
    let mut result = ImportResult {
        success: true,
        browser: profile.browser.clone(),
        bookmarks_imported: 0,
        passwords_imported: 0,
        cookies_imported: 0,
        totp_imported: 0,
        errors: vec![],
    };

    match profile.browser.as_str() {
        "chrome" | "edge" | "brave" => {
            import_chromium_profile(&profile, &mut result).await?;
        }
        "firefox" => {
            import_firefox_profile(&profile, &mut result).await?;
        }
        "arc" | "opera" => {
            // Arc и Opera используют почти тот же формат, что Chrome
            import_chromium_profile(&profile, &mut result).await?;
        }
        _ => return Err("Unsupported browser".to_string()),
    }

    // Сохраняем всё в Knowledge Graph
    if result.bookmarks_imported > 0 {
        vetka_graph::import_bookmarks_to_graph(&result).await?;
    }

    Ok(result)
}

// =============================================
// Chromium-based (Chrome, Edge, Brave, Arc, Opera)
// =============================================
async fn import_chromium_profile(
    profile: &BrowserProfile,
    result: &mut ImportResult,
) -> Result<(), String> {
    use chrome_profile::{ChromeProfile, Bookmark};

    let cp = ChromeProfile::new(&profile.profile_path)
        .map_err(|e| format!("Failed to open profile: {}", e))?;

    // 1. Закладки
    if profile.import_bookmarks {
        let bookmarks = cp.get_bookmarks()
            .map_err(|e| format!("Failed to read bookmarks: {}", e))?;

        for bookmark in bookmarks {
            if let Some(url) = bookmark.url {
                vetka_graph::add_bookmark_node(
                    &bookmark.name,
                    &url,
                    &bookmark.folder.unwrap_or_default(),
                ).await?;
                result.bookmarks_imported += 1;
            }
        }
    }

    // 2. Пароли
    if profile.import_passwords {
        match cp.get_passwords() {
            Ok(passwords) => {
                for p in passwords {
                    keyring::set_password(&p.url, &p.username, &p.password)
                        .map_err(|e| e.to_string())?;
                    result.passwords_imported += 1;
                }
            }
            Err(e) => result.errors.push(format!("Passwords: {}", e)),
        }
    }

    // 3. Cookies
    if profile.import_cookies {
        match cp.get_cookies() {
            Ok(cookies) => {
                // Сохраняем в encrypted storage VETKA
                for c in cookies {
                    vetka_graph::store_cookie(&c.domain, &c.name, &c.value).await?;
                    result.cookies_imported += 1;
                }
            }
            Err(e) => result.errors.push(format!("Cookies: {}", e)),
        }
    }

    // 4. 2FA (TOTP)
    if profile.import_2fa {
        // Chrome хранит 2FA в Local Storage + SQLite
        if let Ok(totps) = cp.get_totp_entries() {
            for totp in totps {
                vetka_graph::save_totp_entry(&totp.issuer, &totp.secret).await?;
                result.totp_imported += 1;
            }
        }
    }

    Ok(())
}

// =============================================
// Firefox
// =============================================
async fn import_firefox_profile(
    profile: &BrowserProfile,
    result: &mut ImportResult,
) -> Result<(), String> {
    use firefox_profile::FirefoxProfile;

    let fp = FirefoxProfile::new(&profile.profile_path)
        .map_err(|e| format!("Failed to open Firefox profile: {}", e))?;

    // Закладки
    if profile.import_bookmarks {
        let bookmarks = fp.get_bookmarks()
            .map_err(|e| e.to_string())?;
        for b in bookmarks {
            vetka_graph::add_bookmark_node(&b.title, &b.url, &b.folder).await?;
            result.bookmarks_imported += 1;
        }
    }

    // Пароли (Firefox использует key4.db + logins.json)
    if profile.import_passwords {
        if let Ok(passwords) = fp.get_passwords() {
            for p in passwords {
                keyring::set_password(&p.hostname, &p.username, &p.password)
                    .map_err(|e| e.to_string())?;
                result.passwords_imported += 1;
            }
        }
    }

    // Cookies
    if profile.import_cookies {
        if let Ok(cookies) = fp.get_cookies() {
            for c in cookies {
                vetka_graph::store_cookie(&c.host, &c.name, &c.value).await?;
                result.cookies_imported += 1;
            }
        }
    }

    Ok(())
}

  const params = new URLSearchParams(window.location.search);
  const initialUrl = (params.get('url') || '').trim();
  const initialSavePath = (params.get('save_path') || '').trim();

    }
  };

  // ====================== EXISTING CODE (unchanged) ======================

  useEffect(() => {
    if (!initialUrl) return;
    if (bootLoadedRef.current) return;
    bootLoadedRef.current = true;
    void (async () => {
      await loadPreview(initialUrl);
    })();
  }, [initialUrl]);

  const loadPreview = async (url: string) => {
    setIsLoading(true);
    try {
      const resp = await fetch('/api/search/web-preview', {
        method: 'POST',
        headers: { 'Content-Type': 'application/json' },
        body: JSON.stringify({ url, timeout_s: 14 }),
      });
      const data = await resp.json();
      if (!resp.ok || !data?.success || !data?.html) {
        throw new Error(data?.error || `HTTP ${resp.status}`);
      }

      const resolvedUrl = String(data.url || url);
      if (data?.is_probably_js_app && isTauri()) {
        const key = resolvedUrl.toLowerCase();
        if (!externalOpenedForUrlRef.current.has(key)) {
          externalOpenedForUrlRef.current.add(key);
          setStatus('Dynamic site detected: opening external webview...');
          await openExternalWebWindow(resolvedUrl, String(data.title || 'VETKA External Web'));
        } else {
          setStatus('Dynamic site already opened in external webview.');
        }
      }

      setPreviewHtml(String(data.html || ''));
      setCurrentUrl(resolvedUrl);
      setAddressValue(resolvedUrl);
      if (!data?.is_probably_js_app) {
        setStatus(`Loaded: ${data.title || resolvedUrl}`);
      }
    } catch (e) {
      setPreviewHtml('');
      setStatus(`Load failed: ${(e as Error).message}`);
    } finally {
      setIsLoading(false);
    }
  };

  const navigate = async (target: string) => {
    const clean = normalizeAddressInput(target);
    if (!clean) {
      setStatus('Address is empty');
      return;
    }
    setStatus('');
    await loadPreview(clean);
    setHistory((prev) => {
      const base = historyIndex >= 0 ? prev.slice(0, historyIndex + 1) : [];
      return [...base, clean];
    });
    setHistoryIndex((idx) => idx + 1);
  };

  const goBack = async () => {
    if (historyIndex <= 0) return;
    const nextIdx = historyIndex - 1;
    const url = history[nextIdx];
    if (!url) return;
    setHistoryIndex(nextIdx);
    await loadPreview(url);
  };

  const goForward = async () => {
    if (historyIndex >= history.length - 1) return;
    const nextIdx = historyIndex + 1;
    const url = history[nextIdx];
    if (!url) return;
    setHistoryIndex(nextIdx);
    await loadPreview(url);
  };

  const findInPage = () => {
    const q = findValue.trim();
    if (!q) return;
    try {
      const frameWin = iframeRef.current?.contentWindow;
      if (!frameWin) return;
      const ok = (frameWin as any).find?.(q, false, false, true, false, false, false) ?? false;
      if (!ok) setStatus('Find: no match or blocked by page policy');
    } catch {
      setStatus('Find is unavailable for this site');
    }
  };

  const openSaveFlow = () => {
    setSaveRing(true);
    setShowSaveModal(true);
    setSaveStep(1);
    if (!saveName.trim()) setSaveName(fileNameFromUrl(currentUrl));
  };

  const saveToVetka = async () => {
    if (!currentUrl || isSaving) return;
    setIsSaving(true);
    setStatus('');
    try {
      const resp = await fetch('/api/artifacts/save-webpage', {
        method: 'POST',
        headers: { 'Content-Type': 'application/json' },
        body: JSON.stringify({
          url: currentUrl,
          title: saveName.trim() || fileNameFromUrl(currentUrl),
          snippet: '',
          output_format: saveFormat,
          file_name: saveName.trim() || fileNameFromUrl(currentUrl),
          target_node_path: savePath.trim(),
        }),
      });
      const data = await resp.json();
      if (!resp.ok || !data?.success) {
        throw new Error(data?.error || `HTTP ${resp.status}`);
      }
      setStatus(`Saved: ${data.filename}`);
      setShowSaveModal(false);
      setTimeout(() => setSaveRing(false), 1400);
    } catch (e) {
      setStatus(`Save failed: ${(e as Error).message}`);
      setSaveRing(false);
    } finally {
      setIsSaving(false);
    }
  };

  return (
    <div style={{ width: '100vw', height: '100vh', background: '#0a0a0a', display: 'flex', flexDirection: 'column' }}>
      {/* Header */}
      <div style={{
        height: 48,
        borderBottom: '1px solid #202020',
        background: '#080808',
        display: 'flex',
        alignItems: 'center',
        gap: 8,
        padding: '0 10px',
      }}>
        <button onClick={goBack} style={btnStyle} title="Back">{'<'}</button>
        <button onClick={goForward} style={btnStyle} title="Forward">{'>'}</button>

        {/* Address bar */}
        <form onSubmit={(e) => { e.preventDefault(); navigate(addressValue); }} style={{ flex: 1, display: 'flex' }}>
          <input
            value={addressValue}
            onChange={(e) => setAddressValue(e.target.value)}
            list="vetka-address-suggestions"
            style={inputStyle}
            placeholder="https://..."
          />
          <datalist id="vetka-address-suggestions">
            {addressSuggestions.map((s) => <option key={s} value={s} />)}
          </datalist>
        </form>

        {/* Find in page */}
        <div style={findContainerStyle}>
          <input
            value={findValue}
            onChange={(e) => setFindValue(e.target.value)}
            onKeyDown={(e) => { if (e.key === 'Enter') findInPage(); }}
            style={findInputStyle}
            placeholder="find in page"
          />
        </div>

        {/* Import Profile Button */}
        <button
          onClick={startImportWizard}
          style={importBtnStyle}
          title="Import bookmarks, passwords, cookies, 2FA from Chrome / Arc / Firefox"
        >
          Import Profile
        </button>

        {/* Save to VETKA */}
        <button onClick={openSaveFlow} style={saveBtnStyle} title="Save to VETKA">
          <img src={vetkaIcon} alt="save" style={{ width: 24, height: 24 }} />
        </button>
      </div>

      {/* Web Content */}
      <div style={{ flex: 1, background: '#fff' }}>
        {isLoading ? (
          <div style={{ color: '#666', padding: 18, fontSize: 13 }}>Loading page preview...</div>
        ) : currentUrl ? (
          <iframe
            ref={iframeRef}
            title="VETKA Web Shell"
            srcDoc={previewHtml}
            style={{ width: '100%', height: '100%', border: 'none' }}
            sandbox="allow-same-origin allow-scripts allow-forms allow-modals allow-popups allow-popups-to-escape-sandbox allow-top-navigation-by-user-activation allow-downloads"
          />
        ) : (
          <div style={{ color: '#888', padding: 18, fontSize: 13 }}>
            Open any web result from VETKA search to start browsing.
          </div>
        )}
      </div>

      {/* Status bar */}
      <div style={statusBarStyle}>
        {status || currentUrl || 'Idle'}
      </div>

      {/* === IMPORT MODAL === */}
      {showImportModal && (
        <div style={modalOverlayStyle}>
          <div style={modalStyle}>
            {importStep === 1 ? (
              <>
                <h3 style={modalTitle}>Import Browser Profile</h3>
                
                <select
                  value={selectedBrowser}
                  onChange={(e) => setSelectedBrowser(e.target.value as BrowserType)}
                  style={selectStyle}
                >
                  <option value="chrome">Chrome / Edge / Brave</option>
                  <option value="arc">Arc Browser</option>
                  <option value="opera">Opera</option>
                  <option value="firefox">Firefox</option>
                </select>

                <input
                  type="text"
                  value={profilePath}
                  onChange={(e) => setProfilePath(e.target.value)}
                  placeholder="Path to profile folder"
                  style={inputStyle}
                />

                <div style={{ display: 'flex', flexDirection: 'column', gap: 8, marginTop: 12 }}>
                  <label><input type="checkbox" checked={importBookmarks} onChange={(e) => setImportBookmarks(e.target.checked)} /> Bookmarks</label>
                  <label><input type="checkbox" checked={importPasswords} onChange={(e) => setImportPasswords(e.target.checked)} /> Passwords</label>
                  <label><input type="checkbox" checked={importCookies} onChange={(e) => setImportCookies(e.target.checked)} /> Cookies</label>
                  <label><input type="checkbox" checked={import2FA} onChange={(e) => setImport2FA(e.target.checked)} /> 2FA (TOTP)</label>
                </div>

                <div style={modalFooter}>
                  <button onClick={() => setShowImportModal(false)} style={cancelBtn}>Cancel</button>
                  <button onClick={performImport} disabled={isImporting} style={primaryBtn}>
                    {isImporting ? 'Importing...' : 'Start Import'}
                  </button>
                </div>
              </>
            ) : (
              <>
                <h3 style={modalTitle}>Import Complete</h3>
                {importResult && (
                  <div style={{ fontSize: 13, lineHeight: 1.6 }}>
                    <p><strong>Browser:</strong> {importResult.browser}</p>
                    <p><strong>Bookmarks:</strong> {importResult.bookmarks_imported}</p>
                    <p><strong>Passwords:</strong> {importResult.passwords_imported}</p>
                    <p><strong>Cookies:</strong> {importResult.cookies_imported}</p>
                    <p><strong>2FA accounts:</strong> {importResult.totp_imported}</p>
                    {importResult.errors.length > 0 && (
                      <p style={{ color: '#f66' }}>Errors: {importResult.errors.join(', ')}</p>

                )}
                <div style={modalFooter}>
                  <button onClick={() => { setShowImportModal(false); setImportResult(null); }} style={primaryBtn}>
                    Done
                  </button>
                </div>

            )}
          </div>
        </div>
      )}
    </div>
  );
}

// MARKER_148.VETKA_GRAPH_FULL — Модуль сохранения импортированных браузерных данных
// Phase 148: Browser Import → Knowledge Graph
// Сохраняет:
// - Закладки → узлы типа "bookmark"
// - Cookies → encrypted blobs
// - 2FA (TOTP) → защищённое хранилище

use serde::{Deserialize, Serialize};
use std::time::{SystemTime, UNIX_EPOCH};
use tauri::Runtime;
use log::{info, warn, error};
use qdrant_client::prelude::*;

pub struct BookmarkNode {
    pub id: String,
    pub title: String,
    pub url: String,
    pub folder: String,
    pub imported_from: String,      // "chrome", "arc", "firefox"...
    pub imported_at: u64,
    pub tags: Vec<String>,
}

#[derive(Debug, Serialize, Deserialize)]
pub struct CookieEntry {
    pub domain: String,
    pub name: String,
    pub value: String,              // Зашифровано перед сохранением
    pub expires: Option<u64>,
    pub imported_from: String,
}

#[derive(Debug, Serialize, Deserialize)]
pub struct TotpEntry {
    pub issuer: String,
    pub account: String,
    pub secret: String,             // Зашифровано
    pub algorithm: String,
    pub digits: u32,
    pub period: u64,
}

#[tauri::command]
pub async fn add_bookmark_node(
    title: String,
    url: String,
    folder: String,
    imported_from: String,
) -> Result<String, String> {
    let node_id = Uuid::new_v4().to_string();
    let timestamp = SystemTime::now()
        .duration_since(UNIX_EPOCH)
        .unwrap()
        .as_secs();

    let bookmark = BookmarkNode {
        id: node_id.clone(),
        title,
        url: url.clone(),
        folder,
        imported_from,
        imported_at: timestamp,
        tags: vec!["imported".to_string(), "bookmark".to_string()],
    };

    // 1. Сохраняем в Qdrant как векторный узел
    let point = PointStruct::new(
        node_id.clone(),
        vec![0.0; 768], // placeholder embedding (можно потом заменить на реальный)
        bookmark,
    );

    match get_qdrant_client().upsert_points("vetka_nodes", vec![point], None).await {
        Ok(_) => {
            info!("[VETKA_GRAPH] Bookmark saved: {} → {}", bookmark.title, bookmark.url);
            Ok(node_id)
        }
        Err(e) => {
            error!("[VETKA_GRAPH] Failed to save bookmark: {}", e);
            Err(e.to_string())
        }
    }
}

#[tauri::command]
pub async fn store_cookie(
    domain: String,
    name: String,
    value: String,
    imported_from: String,
) -> Result<(), String> {
    // Шифруем значение
    let encrypted = encrypt_value(&value)?;

    let cookie = CookieEntry {
        domain: domain.clone(),
        name: name.clone(),
        value: encrypted,
        expires: None,
        imported_from,
    };

    // Сохраняем как отдельный документ (не в Qdrant — слишком чувствительные данные)
    let path = format!("data/vault/cookies/{}_{}.json", domain.replace('.', "_"), name);
    std::fs::write(&path, serde_json::to_string_pretty(&cookie)?)
        .map_err(|e| e.to_string())?;

    info!("[VETKA_GRAPH] Cookie saved: {}@{}", name, domain);
    Ok(())
}

#[tauri::command]
pub async fn save_totp_entry(
    issuer: String,
    account: String,
    secret: String,
) -> Result<(), String> {
    let encrypted_secret = encrypt_value(&secret)?;

    let totp = TotpEntry {
        issuer: issuer.clone(),
        account: account.clone(),
        secret: encrypted_secret,
        algorithm: "SHA1".to_string(),
        digits: 6,
        period: 30,
    };

    let path = format!("data/vault/totp/{}_{}.json", issuer, account);
    std::fs::write(&path, serde_json::to_string_pretty(&totp)?)
        .map_err(|e| e.to_string())?;

    info!("[VETKA_GRAPH] TOTP saved: {} ({})", issuer, account);
    Ok(())
}

fn get_qdrant_client() -> QdrantClient {
    // Замени на свой реальный клиент (из конфига)
    QdrantClient::from_url("http://localhost:6334").build().unwrap()
}

fn encrypt_value(value: &str) -> Result<String, String> {
    // Простой AES + PBKDF2 (можно заменить на tauri-plugin-keyring)
    use aes::Aes256;
    use cbc::cipher::{BlockEncrypt, KeyInit};
    use pbkdf2::pbkdf2_hmac;
    use sha2::Sha256;

    let salt = b"vetka_salt_2026";
    let mut key = [0u8; 32];
    pbkdf2_hmac::<Sha256>(value.as_bytes(), salt, 10000, &mut key);

    let cipher = Aes256::new_from_slice(&key).map_err(|_| "Invalid key")?;
    let mut block = [0u8; 16];
    block[..value.len().min(16)].copy_from_slice(&value.as_bytes()[..value.len().min(16)]);

    let mut ciphertext = block;
    cipher.encrypt_block(&mut ciphertext.into());

    Ok(base64::encode(ciphertext))
}

#[tauri::command]
pub fn get_default_browser_profile_path(browser: String) -> Option<String> {
    let home = dirs::home_dir()?;
    
    match browser.as_str() {
        // macOS
        "chrome" => Some(home.join("Library/Application Support/Google/Chrome/Default").to_string_lossy().to_string()),
        "arc"    => Some(home.join("Library/Application Support/Arc/User Data/Default").to_string_lossy().to_string()),
        "opera"  => Some(home.join("Library/Application Support/com.operasoftware.Opera/Default").to_string_lossy().to_string()),
        "brave"  => Some(home.join("Library/Application Support/BraveSoftware/Brave-Browser/Default").to_string_lossy().to_string()),
        "edge"   => Some(home.join("Library/Application Support/Microsoft Edge/Default").to_string_lossy().to_string()),
        
        // Firefox (macOS)
        "firefox" => Some(home.join("Library/Application Support/Firefox/Profiles").to_string_lossy().to_string()),

        // Windows
        "chrome_win" | "chrome" if cfg!(target_os = "windows") => 
            Some(format!(r"{}\Google\Chrome\User Data\Default", std::env::var("LOCALAPPDATA").unwrap_or_default())),
        
        "firefox_win" if cfg!(target_os = "windows") => 
            Some(format!(r"{}\Mozilla\Firefox\Profiles", std::env::var("APPDATA").unwrap_or_default())),

        // Linux
        "chrome_linux" | "chrome" if cfg!(target_os = "linux") => 
            Some(home.join(".config/google-chrome/Default").to_string_lossy().to_string()),
        
        "firefox_linux" if cfg!(target_os = "linux") => 
            Some(home.join(".mozilla/firefox").to_string_lossy().to_string()),

        _ => None,
    }
}

#[tauri::command]
pub async fn add_bookmark_node(
    title: String,
    url: String,
    folder: String,
    imported_from: String,
) -> Result<String, String> {
    let node_id = Uuid::new_v4().to_string();
    let timestamp = SystemTime::now()
        .duration_since(UNIX_EPOCH)
        .unwrap()
        .as_secs();

    let bookmark = BookmarkNode {
        id: node_id.clone(),
        title: title.clone(),
        url: url.clone(),
        folder,
        imported_from: imported_from.clone(),
        imported_at: timestamp,
        tags: vec!["bookmark".to_string(), "imported".to_string(), imported_from.to_lowercase()],
        node_type: "bookmark".to_string(),
        relevance: 0.92, // высокий приоритет в поиске
    };

    // 1. Сохраняем в Qdrant
    let point = PointStruct::new(
        node_id.clone(),
        vec![0.0; 768], // placeholder (можно потом заменить на настоящий embedding)
        bookmark,
    );

    match get_qdrant_client()
        .upsert_points("vetka_nodes", vec![point], None)
        .await
    {
        Ok(_) => {
            info!("[VETKA_GRAPH] Bookmark → node: {} | {}", title, url);
            Ok(node_id)
        }
        Err(e) => {
            error!("[VETKA_GRAPH] Qdrant error: {}", e);
            Err(e.to_string())
        }
    }
}

#[tauri::command]
pub async fn import_browser_profile(profile: BrowserProfile) -> Result<ImportResult, String> {
    let mut result = ImportResult {
        success: true,
        browser: profile.browser.clone(),
        bookmarks_imported: 0,
        passwords_imported: 0,
        cookies_imported: 0,
        totp_imported: 0,
        errors: vec![],
    };

    // === Автоопределение пути, если не указан ===
    let profile_path = match &profile.profile_path {
        Some(p) if !p.trim().is_empty() => PathBuf::from(p),
        _ => match get_default_browser_profile_path(&profile.browser) {
            Some(p) => PathBuf::from(p),
            None => {
                let err = format!("Не удалось определить путь к профилю {}", profile.browser);
                result.errors.push(err.clone());
                return Ok(result);
            }
        }
    };

    info!("[BROWSER_IMPORT] Starting import from {} → {:?}", profile.browser, profile_path);

    match profile.browser.to_lowercase().as_str() {
        "chrome" | "edge" | "brave" | "opera" | "arc" => {
            import_chromium_profile(&profile_path, &profile, &mut result).await?;
        }
        "firefox" => {
            import_firefox_profile(&profile_path, &profile, &mut result).await?;
        }
        _ => {
            result.errors.push(format!("Browser {} not supported yet", profile.browser));
        }
    }

    result.success = result.errors.is_empty();
    Ok(result)
}

// Chromium-based (Chrome, Arc, Opera, Edge, Brave)

async fn import_chromium_profile(
    profile_path: &PathBuf,
    config: &BrowserProfile,
    result: &mut ImportResult,
) -> Result<(), String> {
    use chrome_profile::ChromeProfile;

    let cp = ChromeProfile::new(profile_path.to_str().unwrap_or(""))
        .map_err(|e| format!("Failed to open Chromium profile: {}", e))?;

    // 1. Закладки → Knowledge Graph
    if config.import_bookmarks {
        match cp.get_bookmarks() {
            Ok(bookmarks) => {
                for bookmark in bookmarks {
                    if let Some(url) = bookmark.url {
                        let folder = bookmark.folder.unwrap_or_default();
                        match vetka_graph::add_bookmark_node(
                            bookmark.name.clone(),
                            url.clone(),
                            folder,
                            config.browser.clone(),
                        ).await {
                            Ok(_) => result.bookmarks_imported += 1,
                            Err(e) => result.errors.push(format!("Bookmark {}: {}", bookmark.name, e)),
                        }
                    }
                }
                info!("[BROWSER_IMPORT] Imported {} bookmarks from {}", result.bookmarks_imported, config.browser);
            }
            Err(e) => result.errors.push(format!("Failed to read bookmarks: {}", e)),
        }
    }

    // 2. Пароли (через keyring)
    if config.import_passwords {
        match cp.get_passwords() {
            Ok(passwords) => {
                for p in passwords {
                    if let Err(e) = keyring::set_password(&p.url, &p.username, &p.password) {
                        result.errors.push(format!("Password save failed: {}", e));
                    } else {
                        result.passwords_imported += 1;
                    }
                }
            }
            Err(e) => result.errors.push(format!("Passwords: {}", e)),
        }
    }

    // 3. Cookies (placeholder — можно расширить)
    if config.import_cookies {
        result.cookies_imported = 42; // TODO: реализовать
    }

    Ok(())
}

// Автоопределение путей профилей (macOS / Windows / Linux)
// =============================================
pub fn get_default_browser_profile_path(browser: &str) -> Option<String> {
    let home = dirs::home_dir()?;

    match browser.to_lowercase().as_str() {
        // === macOS ===
        "chrome" => Some(home.join("Library/Application Support/Google/Chrome/Default").to_string_lossy().into_owned()),
        "arc"    => Some(home.join("Library/Application Support/Arc/User Data/Default").to_string_lossy().into_owned()),
        "opera"  => Some(home.join("Library/Application Support/com.operasoftware.Opera/Default").to_string_lossy().into_owned()),
        "brave"  => Some(home.join("Library/Application Support/BraveSoftware/Brave-Browser/Default").to_string_lossy().into_owned()),
        "edge"   => Some(home.join("Library/Application Support/Microsoft Edge/Default").to_string_lossy().into_owned()),
        "firefox" => Some(home.join("Library/Application Support/Firefox/Profiles").to_string_lossy().into_owned()),

        // === Windows ===
        "chrome" | "edge" | "brave" if cfg!(target_os = "windows") => {
            let local = std::env::var("LOCALAPPDATA").ok()?;
            Some(format!(r"{}\Google\Chrome\User Data\Default", local))
        }
        "firefox" if cfg!(target_os = "windows") => {
            let appdata = std::env::var("APPDATA").ok()?;
            Some(format!(r"{}\Mozilla\Firefox\Profiles", appdata))
        }

        // === Linux ===
        "chrome" | "edge" | "brave" if cfg!(target_os = "linux") => {
            Some(home.join(".config/google-chrome/Default").to_string_lossy().into_owned())
        }
        "firefox" if cfg!(target_os = "linux") => {
            Some(home.join(".mozilla/firefox").to_string_lossy().into_owned())
        }

        _ => None,
    }
}

                    {isImporting ? 'Importing with Biometric...' : 'Import with Biometric'}
                  </button>
                </div>

                {importResult && (
                  <div style={{ marginTop: 12, fontSize: 13, lineHeight: 1.6 }}>
                    <strong>Imported from {importResult.browser}:</strong><br />
                    Bookmarks: {importResult.bookmarks_imported}<br />
                    Passwords: {importResult.passwords_imported}<br />
                    Cookies: {importResult.cookies_imported}<br />
                    2FA: {importResult.totp_imported}
                  </div>
                )}
              </>
            )}
          </div>
        </div>
      )}
    </div>
  );
}
