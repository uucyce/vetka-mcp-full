## 1. Voice Assistant Integration Research

### Models Compared for M4 Mac

| Model | Type | Latency | Quality | M4 Optimized |
|-------|------|---------|---------|--------------|
| **Qwen3-TTS** | TTS | ~200ms | High | ✓ Metal |
| **Personaplex-7b-v1** | Voice Clone | ~500ms | Very High | Partial |
| **Whisper** | STT | ~300ms | Excellent | ✓ Metal |

### Grok's Recommendation (from group chat)
> "Qwen быстрее" - Qwen3-TTS recommended for real-time voice

### Proposed Architecture

```
User Voice → Whisper (STT) → VETKA LLM → Qwen3-TTS (TTS) → Audio Output
                ↓                              ↓
         ~300ms latency              ~200ms latency

Total target: <1 second response
```

### Implementation Options

**Option A: Local Pipeline (Recommended for M4)**
```python
# Using MLX for Metal acceleration
from mlx_whisper import transcribe
from qwen_tts import synthesize

async def voice_pipeline(audio_input):
    # STT
    text = await transcribe(audio_input, model="whisper-large-v3")

    # LLM Response
    response = await vetka_llm.generate(text)

    # TTS
    audio = await synthesize(response, voice="vetka_default")
    return audio
```

**Option B: Hybrid (API fallback)**
```python
# Local STT, API for complex TTS
whisper_local = True
tts_api_fallback = "elevenlabs"  # For voice cloning
```

## 2. Pipeline Progress Streaming

### Implemented (MARKER_102.26-28)

Progress streaming from agent_pipeline.py to Vetka chat:

```python
# In agent_pipeline.py
async def _stream_progress_to_vetka(self, chat_id: str, message: str):
    """Stream progress updates to Vetka chat"""
    if self.mcp_bridge:
        await self.mcp_bridge.send_progress(chat_id, message)

# Usage in execute()
await self._stream_progress_to_vetka(
    chat_id,
    f"📊 Subtask {i+1}/{total}: {subtask.description}"
)
```

### Streaming Events

| Event | Format | When |
|-------|--------|------|
| Task Start | `🚀 Pipeline: {task_name}` | On execute() |
| Subtask Progress | `📊 Subtask {n}/{total}: {desc}` | Each subtask |
| Tool Call | `🔧 Tool: {tool_name}` | Optional |
| Completion | `✅ Pipeline complete` | On success |
| Error | `❌ Error: {message}` | On failure |
