[DEBRIEF-BUG] Analysis: Scrub Sync Throttling Issue
Problem Summary
The bug describes a dual-path synchronization problem in the video NLE (Non-Linear Editor):
Scrubber sync fires on every rAF frame (60x/sec) during playback
0.04s (40ms) throttle prevents spam but may cause visual flicker with short clips
Root cause: The same sync logic handles both user scrubbing AND playback progression without distinguishing the context
Why 40ms Throttle Is Problematic
Table
Scenario	Issue
24fps footage	40ms throttle ≈ 1.5 frames — acceptable
60fps footage	40ms throttle ≈ 2.4 frames — visible stutter
Short clips (<1s)	Position updates feel "jumpy" in Source Monitor
High refresh displays (120Hz+)	Throttle creates 2:1 or 3:1 update mismatch
According to research, throttling rAF without proper frame alignment can desynchronize updates from the display refresh, defeating rAF's purpose . The 40ms threshold (25fps) is below the perceptual flicker fusion threshold for many users, especially with high-contrast video content.

Solution B: Frame-Aligned Throttling (Alternative)
If maintaining single sync path is required, use frame-aligned throttling instead of fixed 40ms:

// MARKER_DEBRIEF.BUG1: Frame-aligned throttle prevents flicker

class FrameAlignedThrottle {
  private lastFrameTime = 0;
  private targetFPS = 30; // Match typical video playback UI rate
  private interval = 1000 / this.targetFPS;
  
  shouldUpdate(timestamp: number): boolean {
    const elapsed = timestamp - this.lastFrameTime;
    
    // Allow update if we've passed the interval threshold
    if (elapsed >= this.interval) {
      // Align to multiples of interval to prevent jitter
      this.lastFrameTime = timestamp - (elapsed % this.interval);
      return true;
    }
    return false;
  }
}

// Usage in rAF loop:

const ScrubberThumb = React.memo(({ position, isPlaying }) => {
  const thumbRef = useRef<HTMLDivElement>(null);
  
  useEffect(() => {
    // Direct DOM manipulation bypasses React render cycle
    if (thumbRef.current) {
      thumbRef.current.style.transform = `translateX(${position}px)`;
    }
  }, [position]);
  
  return (
    <div 
      ref={thumbRef}
      className="scrubber-thumb"
      // Use will-change only during playback
      style={{ willChange: isPlaying ? 'transform' : 'auto' }}
    />
  );
});
