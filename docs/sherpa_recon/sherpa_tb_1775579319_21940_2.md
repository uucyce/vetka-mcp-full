def export_parallax_package(
    source_image_path: str,
    output_dir: Optional[str] = None,
    config: Optional[dict] = None,
    generate_motion: bool = True,
) -> str:
    """
    Headless export of full parallax package.

    Args:
        source_image_path: Path to input image (JPEG/PNG)
        output_dir: Where to create the package (default: auto-generated)
        config: Optional overrides for depth, segmentation, inpainting, motion
        generate_motion: If True, create motion_presets.json

    Returns:
        Path to the created package directory
    """

What it does (no UI):

Load image, compute global depth (MiDaS/DPT)

Run layer extraction (Mask2Former + refinement) → binary masks per object

For each mask:

Extract RGBA layer (foreground only)

Extract per‑layer depth (global depth masked)

Store mask as separate PNG

Generate clean plate: combine all foreground masks → inpaint with LaMa → save as clean_plate.png

Build manifest with:

Layer order, z, depthPriority (derived from global depth at mask centroid)

Relative paths to all assets

Camera / lens defaults (overridable via config)

If generate_motion: compute simple motion presets (push‑in, push‑out, pan‑x, pan‑y, orbit) based on depth distribution and layer order. Save as motion_presets.json.

Write everything to output_dir (created if missing). Return absolute path.
