[DEBRIEF-BUG] Analysis: Spec Files Self-Spawning Vite on Individual Ports

The globalSetup is described as "additive" — meaning specs can opt-in via VETKA_GLOBAL_ORIGIN env var, but this is not breaking existing functionality.
