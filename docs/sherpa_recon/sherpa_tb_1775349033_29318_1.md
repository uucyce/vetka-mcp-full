Fix: Disable Q3 Auto-Task Creation in smart_debrief
Problem

smart_debrief._create_auto_tasks() creates [DEBRIEF-IDEA] tasks from Q3 recommendations.
This violates the requirement that Q3 ideas must go only to memory (ENGRAM/CORTEX) and never to the task board.

Solution

Gate Q3 task creation behind a configuration flag ENABLE_Q3_TASK_CREATION = False (default off).

The Q3 loop is kept but never executes unless the flag is manually set to True.

Memory routing (_route_to_memory) remains fully active for Q3 answers.

Update tests to verify no tasks are created for Q3.

def _create_auto_tasks(
    recommendations: list[str],   # Q3 answers
    lessons: list[str],           # Q1 answers
    task_board,
    source: str,
) -> int:
    """Create tasks from Q1 (bugs) and optionally Q3 (ideas)."""
    created = 0

    # Q1: Bugs → always create tasks
    for lesson in lessons:
        if not lesson.strip():
            continue
        summary = _extract_summary(lesson)
        if _is_bug_report(lesson):
            task_id, is_new = _dedup_or_create(
                prefix="DEBRIEF-BUG",
                summary=summary,
                full_text=lesson,
                task_board=task_board,
                phase_type="bug",
                priority=1,
                tags=["bug", "auto", source],
                source=source,
                description=f"Auto‑created from debrief bug report:\n{lesson}",
            )
            if is_new:
                created += 1
        else:
            # Non‑bug lesson → still create as research task
            task_id, is_new = _dedup_or_create(
                prefix="DEBRIEF-LESSON",
                summary=summary,
                full_text=lesson,
                task_board=task_board,
                phase_type="lesson",
                priority=2,
                tags=["lesson", "auto", source],
                source=source,
                description=f"Auto‑created from debrief lesson:\n{lesson}",
            )
            if is_new:
                created += 1

    # Q3: Ideas → memory only (task creation disabled by default)
    if ENABLE_Q3_TASK_CREATION:
        for rec in recommendations:
            if not rec.strip():
                continue
            summary = _extract_summary(rec)
            task_id, is_new = _dedup_or_create(
                prefix="DEBRIEF-IDEA",
                summary=summary,
                full_text=rec,
                task_board=task_board,
                phase_type="idea",
                priority=3,
                tags=["idea", "auto", source],
                source=source,
                description=f"Auto‑created from debrief recommendation:\n{rec}",
            )
            if is_new:
                created += 1
    else:
        logger.info("Q3 idea task creation disabled – routing only to memory")

    return created
