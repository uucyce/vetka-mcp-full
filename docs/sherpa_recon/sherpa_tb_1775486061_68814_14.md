GEMMA-PHASE1: Gemma 4 as Full Coding Agent — Research Summary
Executive Summary
Testing Gemma 4 (e4b) as a Claude-independent coding agent is viable through OpenCode + Ollama. The setup requires specific configuration for context windows, tool calling support, and authentication placeholders. Alternative tools (free-code, aider) are available if OpenCode proves insufficient.
1. OpenCode + Ollama Configuration
Step 1: Install and Configure Ollama
bash
Copy
# Install Ollama
brew install --cask ollama

# Pull Gemma 4 e4b (9.6GB, fits 16GB+ Macs)
ollama pull gemma4:e4b

# Create 32K context variant (critical for coding tasks)
ollama run gemma4:e4b
/set parameter num_ctx 32768
/save gemma4:e4b-32k
/bye
Critical: Default Ollama context is 4K, which causes issues with coding agents. The model constantly asks "Just let me know what you want to do" because system prompts consume the entire context window .

2. Spawn Command for VETKA Project
Based on the target paths (scripts/spawn_synapse.sh, scripts/synapse_write.sh), here's the documented spawn command:
bash
Copy
#!/bin/bash
# scripts/spawn_synapse.sh - Gemma 4 Agent Spawn

WORKTREE_PATH="${1:-./polaris-lambda}"
TASK_BOARD="${2:-data/templates/agent_registry.yaml}"

# Ensure Ollama is running with extended context
OLLAMA_CONTEXT_LENGTH=32768 OLLAMA_KEEP_ALIVE="-1" ollama serve &

# Wait for Ollama to be ready
sleep 3

# Verify model is available
if ! ollama list | grep -q "gemma4:e4b-32k"; then
    echo "Creating 32K context variant..."
    ollama run gemma4:e4b &
    sleep 5
    echo "/set parameter num_ctx 32768" | ollama run gemma4:e4b
    echo "/save gemma4:e4b-32k" | ollama run gemma4:e4b
    echo "/bye" | ollama run gemma4:e4b
fi

# Spawn OpenCode agent in worktree
cd "$WORKTREE_PATH" || exit 1

# Initialize if AGENTS.md doesn't exist
if [[ ! -f "AGENTS.md" ]]; then
    opencode run "/init"
fi

# Launch TUI with Gemma 4
opencode --model ollama/gemma4:e4b-32k

# Register agent in task board
AGENT_ID="gemma4-$(date +%s)"
echo "- id: $AGENT_ID" >> "$TASK_BOARD"
echo "  model: gemma4:e4b-32k" >> "$TASK_BOARD"
echo "  provider: ollama" >> "$TASK_BOARD"
echo "  worktree: $WORKTREE_PATH" >> "$TASK_BOARD"
echo "  spawned_at: $(date -Iseconds)" >> "$TASK_BOARD"
