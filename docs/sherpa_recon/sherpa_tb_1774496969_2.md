Analysis
Aspect	Details
Tesseract weakness	Mixed scripts (Cyrillic + Latin) cause character misclassification, especially on low‑contrast or dark backgrounds.
Vision framework strength	Native support for multiple scripts in one image, good contrast handling, and macOS/iOS acceleration.
Deployment constraint	Vision framework is Apple‑only (macOS, iOS). VETKA’s FastAPI backend may run on Linux servers (e.g., Ubuntu, Alpine).
Alternatives	Cross‑platform mixed‑script OCR: EasyOCR (PyTorch, supports 80+ languages), PaddleOCR (high accuracy), Google Cloud Vision API (paid), Azure Computer Vision (paid).
