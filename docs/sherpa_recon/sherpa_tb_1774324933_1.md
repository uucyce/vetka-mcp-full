DeprecationWarning: The current event loop policy is the default one, but the asyncio event loop is not running. This is deprecated and will raise an error in the future.

Root cause – The test suite uses pytest-asyncio with an older pattern that obtains the event loop via asyncio.get_event_loop() inside a fixture or test function outside an already‑running loop. Starting with pytest-asyncio>=0.23, the default loop policy behaviour has changed; calling get_event_loop() when no loop is running is deprecated and will become an error.
