2. Approach

The strategy is to introduce a new "focus mode" that makes it explicit whether the user is editing the depth map or previewing the composite parallax effect. This resolves the duplication by assigning a single, primary role to each panel depending on the selected mode.

Step-by-Step Implementation Plan:

Introduce previewFocusMode State:

In App.tsx, create a new React state variable: const [previewFocusMode, setPreviewFocusMode] = useState<'depth' | 'stage'>('stage');.
This state will control which UI area is "active" and will be the single source of truth for the previewing context.

Create a UI Toggle:

Add a new control, such as a segmented button or toggle switch, that is always visible in the main UI.
This control will have two options: "Edit Depth" (sets previewFocusMode to 'depth') and "Preview Parallax" (sets previewFocusMode to 'stage').
This makes the user's intent clear and directly manipulates the new state.

Conditionally Update the Main Stage:

Modify the component that renders the main preview area (Stage).
When previewFocusMode is 'depth', the Stage should only display the static, original source image. It acts as a passive reference. Interaction with it (like camera motion) should be disabled.
When previewFocusMode is 'stage', the Stage should render the full, interactive parallax effect as it currently does.

Apply Distinct CSS Styling:

In index.css, create CSS classes that are applied to a root element based on the previewFocusMode. For example: .preview-focus-depth and .preview-focus-stage.
In depth focus mode:
Visually de-emphasize the main Stage. Apply styles like opacity: 0.6; and pointer-events: none; to make it clear it is not the primary interaction surface.
Visually highlight the Depth panel. Apply a subtle border or box-shadow to draw the user's attention.
In stage focus mode:
The Stage should be fully opaque and interactive.
The Depth panel returns to its default appearance.

Solidify the Depth Panel's Role:

As per the architecture docs, the Depth panel must be the "primary truth surface" for depth information.

Depth Mode: The user's focus is entirely on the Depth panel to adjust the B/W data, using the main Stage only as a static reference.
