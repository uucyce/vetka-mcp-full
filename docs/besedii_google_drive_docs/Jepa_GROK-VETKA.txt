## 1. **I-JEPA = JEPA + VETKA: Твой "Третий Путь" Уже Есть!**
- **Стандарт JEPA**: Context encoder (embed_x) → маска шума → predict target encoder (embed_y). **Не генерирует пиксели/текст** — предсказывает **семантику будущего** (игнор низкоуровневого шума).
- **I-JEPA в VETKA** (твоя интуиция 100%):
  ```

       ↓ Маска шума (ELISION: отсекаем weak/low-confidence edges)
       ↓ Predict L1 (SCC DAG: предсказываем "сверхузлы" + новые связи)
       ↓ L2 View (Knowledge Mode: чистый DAG для UI)
  ```
