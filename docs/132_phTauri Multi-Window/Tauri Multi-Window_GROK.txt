## **1. Research Summary: Tauri Multi-Window + Native Features**

| Feature | Tauri Support | Stability | Cross-Platform |
|---------|---------------|-----------|----------------|
| **Multi-Window** | ✅ Native (v1: `tauri::Builder::new().setup(|app| { app.get_window("main").unwrap().open_devtools(); WindowBuilder::new(app, "devpanel", ...).build()?; })`) | 100% (no Electron overhead) | macOS/Windows/Linux |
| **Always-On-Top** | ✅ `window.setAlwaysOnTop(true)` (JS) или `tauri::window::AlwaysOnTop::true` (Rust) | 99% (rare macOS permission issues) | ✅ (macOS: needs Info.plist) |
| **Notifications** | ✅ `tauri-plugin-notification` (native: NSUserNotification/macOS, Windows Toast, Linux libnotify) | 100% (plugin mature, 1M+ dl) | ✅ (Linux: `notify-osd` dep) |
| **Dev Panel UX** | Floating resizable, transparent BG, badges (titlebar), hotkeys (Cmd+Shift+D) | Prod-ready (e.g., VSCode panels) | ✅ Frameless option |

**Pros vs Alternatives**:
- **Tauri > Electron**: 5MB vs 100MB RAM, true native windows.
- **Polling vs Event-Driven**: Для autonomy — WebSocket (уже в VETKA) + Tauri events (stable).
- **Self-Improving Loop**: DevPanel может poll `/api/tasks` (Phase 131 C20C) + notifs on claim/complete.

**Cons**: v1 → v2 migration soon (multi-window same). macOS always-top needs entitlements.

**Best Patterns** (из modern multi-agent: LangGraph, CrewAI + Tauri):
- **Window Lifecycle**: Create on-demand (command `open_devpanel`), persist state (LocalStorage).
- **BMAD Integration**: Build (fetch tasks), Measure (logs), Analyze (eval_agent.py via API), Decide (notifications).

## **2. Step-by-Step Implementation Guide (30min → Prod)**

### **Prep (5min)**
1. **Install Notification Plugin** (если нет):
   ```bash
   cd client/
   yarn add @tauri-apps/plugin-notification  # или npm
   npx tauri add notification  # Rust side
   yarn tauri dev  # Test
   ```
2. **Update `src-tauri/tauri.conf.json`** (multi-window config):
   ```json
   {
     "windows": [
       {
         "label": "main",  // Existing main window
         "width": 1200, "height": 800
       },
       {
         "label": "devpanel",  // NEW: Dev Panel (hidden by default)
         "width": 400, "height": 600,
         "decorations": false,  // Frameless (floating feel)
         "alwaysOnTop": true,   // Always top!
         "resizable": true,
         "visible": false       // Hidden → open via command
       }
     ],
     "plugins": {
       "notification": { "defaultPermission": "granted" }  // Auto-allow
     }
   }
   ```

### **Rust Backend: src-tauri/src/main.rs (NEW commands, ~20 lines)**
```rust
// MARKER_132_DEV: Tauri DevPanel + Notifications (Phase 132)
#![cfg_attr(not(debug_assertions), windows_subsystem = "windows")]

use tauri::{Manager, WindowBuilder, WindowUrl};
use tauri_plugin_notification::{Notification, NotificationExt};

fn main() {
    tauri::Builder::default()
        .plugin(tauri_plugin_notification::init())  // Plugin init
        .invoke_handler(tauri::generate_handler![open_devpanel, send_dev_notification])
        .setup(|app| {
            // Optional: Auto-open on startup (for dev)
            #[cfg(debug_assertions)]
            open_devpanel(app.app_handle().clone(), None).unwrap();
            Ok(())
        })
        .run(tauri::generate_context!())
        .expect("error while running tauri");
}

// CMD: Open DevPanel (stable, on-demand)
#[tauri::command]
fn open_devpanel(app: tauri::AppHandle, _payload: Option<String>) -> Result<(), String> {
    let window = app.get_webview_window("devpanel")
        .unwrap_or_else(|| {
            WindowBuilder::new(&app, "devpanel", WindowUrl::App("devpanel.html".into()))
                .title("VETKA DevPanel 🌿")
                .always_on_top(true)
                .build()
                .unwrap()
        });
    window.show().unwrap();
    window.set_focus().unwrap();
    Ok(())
}

// CMD: Send Notification (for Mycelium events)
#[tauri::command]
fn send_dev_notification(title: String, body: String) -> Result<(), String> {
    Notification::new(&tauri::Env::default().unwrap(), &title.replace(":", ""))  // Sanitize ID
        .title(&title)
        .body(&body)
        .icon("assets/vetka-icon.png")  // Your icon
        .show()
        .map_err(|e| e.to_string())?;
    Ok(())
}
```
- **Stability**: `unwrap_or_else` prevents crashes. Persist via `window.emit("event", payload)`.

### **Frontend: client/src/DevPanel.tsx (NEW component, ~50 lines)**
Создай `client/src/components/DevPanel.tsx` (iframe-like, loads `/devpanel.html` или React):
```tsx
// MARKER_132_DEV: VETKA DevPanel UI (React + Tauri)
import { useEffect, useState } from 'react';
import { invoke } from '@tauri-apps/api/tauri';
import { listen } from '@tauri-apps/api/event';  // For WS-like events

export default function DevPanel() {
  const [logs, setLogs] = useState<string[]>([]);
  const [tasks, setTasks] = useState<number>(0);

  useEffect(() => {
    // Poll /api/tasks (Phase 131 C20C) every 10s
    const interval = setInterval(async () => {
      const res = await fetch('/api/tasks?status=pending');
      const data = await res.json();
      setTasks(data.length);
      if (data.length > 0) {
        await invoke('send_dev_notification', {
          title: '🛠️ New Tasks!',
          body: `${data.length} pending for Dragon`
        });
      }
    }, 10000);

    // Listen Tauri events (e.g., from Rust/heartbeat)
    const unlisten = listen('heartbeat_tick', (event) => {
      setLogs(prev => [...prev, `Heart: ${event.payload}`]);
      invoke('send_dev_notification', {
        title: '💓 Heartbeat',
        body: 'Tasks checked'
      });
    });

    return () => { clearInterval(interval); unlisten.then(f => f()); };
  }, []);

  return (
    <div className="w-full h-full bg-black/80 text-green-400 p-4 font-mono text-sm overflow-auto">
      <h1>🌿 VETKA DevPanel (Always-Top)</h1>
      <div>Pending Tasks: <span className="badge bg-blue-500">{tasks}</span></div>
      <div>Logs:</div>
      <ul>{logs.slice(-10).map((log, i) => <li key={i}>{log}</li>)}</ul>
      <button onClick={() => invoke('send_dev_notification', {title: 'Test', body: 'Works!'})}>
        Test Notif
      </button>
    </div>
  );
}
```
- **HTML fallback**: Создай `client/public/devpanel.html` с `<div id="app"></div>` + bundle React.
