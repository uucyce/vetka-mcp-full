🎯 THE PROBLEM (Что ломается)
─────────────────────────────────────────────────────────────────────────────
Пользователь пинит файлы → AI получает ОБРЕЗАННЫЙ контекст (~800-1200 токенов)
- Каждый файл обрезается на 3000 символах (CHARACTER truncation, не tokens!)
- Обрезка происходит ПОСЕРЕДИНЕ функции / JSON / класса
- Структура и синтаксис ломаются
- Потеря информации неизбежна
