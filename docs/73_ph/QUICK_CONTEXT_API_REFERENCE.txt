File: src/scanners/dependency_calculator.py
────────────────────────────────────────────────────────────────────────────────

MAIN CLASS:
  DependencyCalculator(config=None, semantic_provider=None)

API:
  calculate(input_data: ScoringInput) -> ScoringResult
    Single A→B dependency score calculation

  calculate_batch(inputs: List[ScoringInput]) -> List[ScoringResult]
    Multiple calculations in order (no aggregation)

  find_dependencies(
      target_file: FileMetadata,
      candidate_sources: List[FileMetadata],
      import_scores: Dict[str, float] = None,
      semantic_scores: Dict[str, float] = None,
      reference_paths: List[str] = None
  ) -> List[ScoringResult]
    Find all significant dependencies for a file
    Returns sorted by score descending, only is_significant=True (>= 0.6)

INPUT DATA STRUCTURES:

  FileMetadata:
    path: str
    created_at: Optional[datetime] = None
    modified_at: Optional[datetime] = None
    rrf_score: float = 0.5
    has_references: bool = False

  ScoringInput:
    source_file: FileMetadata
    target_file: FileMetadata
    import_confidence: float = 0.0       # From PythonScanner
    semantic_score: float = 0.0          # From Qdrant
    has_explicit_reference: bool = False

OUTPUT DATA STRUCTURE:

  ScoringResult:
    source_path: str
    target_path: str
    raw_score: float              # Before sigmoid
    final_score: float            # 0-1 normalized
    is_significant: bool          # >= 0.6
    components: Dict[str, float]  # Debug info
    to_dependency() -> Dependency

FORMULA (Kimi K2, Phase 72.5):

  DEP(A→B) = σ( w₁·I + w₂·S'·E(ΔT) + w₃·R + w₄·RRF )

  where:
    I              = import_confidence (0-1)
    S'             = max(0, (S - 0.5) / 0.5)  # Semantic gating
    E(ΔT)          = 0.2 + 0.8·e^(-ΔT/τ)      # Temporal decay with floor
    R              = 1.0 if explicit_ref else 0.0
    RRF            = source importance (0-1)
    w = [0.4, 0.33, 0.2, 0.07]  (sum = 1.0)
    σ(x) = 1/(1+e^(-12(x-0.35)))  # Sigmoid, center 0.35

  Score interpretation:
    0.8-1.0: Strong dependency (explicit import)
    0.6-0.8: Significant link (semantic + temporal)
    0.4-0.6: Weak link (semantic only)
    0.0-0.4: No dependency

STATUS: ✅ STABLE (just made yesterday, formula 6-AI validated)
