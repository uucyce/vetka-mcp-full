Source Badge Logic:
  result.source === 'qdrant'   → [QD]
  result.source === 'weaviate' → [WV]
  result.source === 'hybrid'   → [HYB]
  result.source === 'other'    → [OTH]
  result.source === undefined  → (no badge)

Active Mode Badge Logic:
  searchMode === 'hybrid'   → [HYB]
  searchMode === 'semantic' → [SEM]
  searchMode === 'keyword'  → [KEY]
  searchMode === 'filename' → [FILE]

Sort Options:
  - Relevance (default descending)
  - Name (A-Z or Z-A)
  - Type (file/code/doc)
  - Date (newest/oldest)
  - Size (largest/smallest)
