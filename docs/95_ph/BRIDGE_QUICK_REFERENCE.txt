❌ Bridge endpoints not found?
   → Check: OPENCODE_BRIDGE_ENABLED=true in environment
   → Restart: python main.py

❌ MCP tool returns connection error?
   → Check: Is VETKA server running? (curl http://localhost:5001/api/health)
   → Check: MCP bridge configuration (VETKA_BASE_URL, VETKA_TIMEOUT)
   → Restart: MCP server in Claude Code/Desktop

❌ All OpenRouter keys rate limited?
   → Automatic rotation will try next key
   → Check: stats endpoint for rate_limited_keys count
   → Solution: Add more keys to OPENCODE_BRIDGE_ENABLED rotation pool

⚠️ Write tool returns dry_run preview?
   → Normal behavior: set dry_run=false to actually execute
   → Example: { dry_run: false } in tool arguments
