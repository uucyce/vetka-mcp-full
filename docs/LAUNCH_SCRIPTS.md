# 🚀 VETKA Live 0.3 - Скрипты запуска

## 🎯 Выберите подходящий способ запуска:

### 1. 🐍 Универсальный Python лаунчер (рекомендуется)
```bash
python start.py
```
**Подходит для:** любой ОС, автоматически определяет платформу

### 2. 🖥️ macOS/Linux - быстрый запуск
```bash
./quick_start.sh
```
**Подходит для:** macOS, Linux, Unix-системы

### 3. 🪟 Windows - пакетный файл
```cmd
run_vetka.bat
```
**Подходит для:** Windows, двойной клик для запуска

### 4. 🔧 Полная настройка (Unix)
```bash
./run_vetka.sh
```
**Подходит для:** macOS, Linux с полной диагностикой

### 5. 🐍 Универсальный Python скрипт
```bash
python launch_vetka.py
```
**Подходит для:** любая ОС, максимальная совместимость

## 🎮 Что делают скрипты:

### ✅ Автоматически:
- Проверяют Python 3.8+
- Создают виртуальное окружение `.venv`
- Устанавливают Python зависимости из `requirements.txt`
- Устанавливают frontend зависимости (если есть Node.js)
- Создают `.env` файл с настройками
- Запускают Weaviate в Docker (если Docker есть)
- Запускают Ollama и загружают модели
- Запускают VETKA на http://localhost:5000

### 🎯 Результат:
- **Готовый к работе VETKA** с 3D визуализацией
- **Автоматическая настройка** всех компонентов
- **Проверка здоровья** системы
- **Интерактивные команды** для тестирования

## 🐛 Если что-то не работает:

### Docker не найден:
```bash
# Установить Docker Desktop
# Или запустить Weaviate вручную:
docker run -p 8080:8080 semitechnologies/weaviate:latest
```

### Ollama не найден:
```bash
# Установить Ollama
curl -fsSL https://ollama.ai/install.sh | sh
ollama serve
```

### Node.js не найден:
```bash
# Установить Node.js
# Frontend будет работать через CDN
```

### Python зависимости:
```bash
pip install -r requirements.txt
```

## 🎉 После успешного запуска:

1. **Откройте браузер**: http://localhost:5000
2. **Попробуйте команды**:
   - `/bot/create API`
   - `/bot/analyze code`
   - `/visual/tree`
3. **Используйте горячие клавиши**:
   - `/` - фокус на ввод
   - `1-5` - смена режимов
   - `g/t/l` - уровни масштаба
4. **Кликайте узлы** для открытия артефактов
5. **Наслаждайтесь 3D деревом знаний!** 🌳✨
