## 🎯 ГЛАВНЫЕ МАРКЕРЫ ПРОБЛЕМ

### 🔴 MARKER #1: MISSING_GET_KEYS_ENDPOINT
**Файл**: `src/api/routes/config_routes.py`
**Проблема**: Нету `@router.get("/keys")` endpoint
**Влияние**: UI не может загрузить список ключей
**Решение**: Добавить ~50 строк кода

### 🔴 MARKER #2: ENDPOINT_FORMAT_MISMATCH
**Файл**: Все routes
**Проблема**: Разные форматы хранения ключей (string, dict, array)
**Влияние**: Endpoint должен нормализовать все форматы
**Решение**: В коде эндпоинта обработать все 4 варианта

### 🔴 MARKER #3: DATA_MASKING_NEEDED
**Файл**: GET /api/keys response
**Проблема**: Не должны возвращаться полные значения ключей
**Влияние**: Security issue если вернуть полные ключи
**Решение**: Mask функция `tvly-dev-...eM9F`
