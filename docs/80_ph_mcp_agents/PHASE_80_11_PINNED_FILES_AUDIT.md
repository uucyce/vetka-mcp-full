## 3. DEEPSEEK ERROR: MODEL LIMITATION

### Ошибка
```
registry.ollama.ai/library/deepseek-llm:7b does not support tools (status code: 400)
```

### Причина
- Deepseek 7B - lightweight модель БЕЗ поддержки tool calling
- `chat_handler.py` (строки 209-213) передаёт `tools=tools` в Ollama
- Deepseek-llm:7b отклоняет запрос

### Код проблемы
```python
# chat_handler.py:209-213
ollama_response = await loop.run_in_executor(
    None,
    lambda: ollama.chat(
        model=model_name,
        messages=messages,
        tools=tools,  # ← FAILS для deepseek-llm:7b
        stream=False
    )
)
```

### Решения
1. Использовать более мощную модель (`deepseek-r1:14b`)
2. Отключить tools для lightweight моделей
3. Использовать OpenRouter API вместо локального Ollama
