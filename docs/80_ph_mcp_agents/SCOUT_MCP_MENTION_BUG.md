## РЕШЕНИЕ (Phase 80.13)

Добавлен отдельный механизм routing для MCP агентов:

1. **MCP_AGENTS registry** - словарь известных MCP агентов с алиасами
2. **notify_mcp_agents()** - функция уведомления через socket + API
3. **Socket event `mcp_mention`** - real-time notification
4. **API endpoint `/api/debug/mcp/mentions/{agent_id}`** - polling

### Новый Flow

```
User: "@browser_haiku помоги"
    ↓
group_message_handler парсит mentions
    ↓
notify_mcp_agents() находит MCP агента
    ↓
Emit socket "mcp_mention"
    ↓
Browser extension получает event
    ↓
Browser Haiku отвечает в чат
```
