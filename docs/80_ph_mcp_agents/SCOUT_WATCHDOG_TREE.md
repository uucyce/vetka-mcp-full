## ROOT CAUSE: ДВА РАЗРЫВА

### 1. Backend `_emit()` метод сломан

**Файл:** `src/scanners/file_watcher.py:382-395`

```python
def _emit(self, event_name: str, data: Dict) -> None:
    if hasattr(self.socketio, 'emit'):
        import asyncio
        try:
            loop = asyncio.get_event_loop()
            if loop.is_running():
                asyncio.create_task(self.socketio.emit(event_name, data))  # ❌ WRONG CONTEXT
            else:
                loop.run_until_complete(self.socketio.emit(event_name, data))  # ❌ BLOCKS
        except RuntimeError:
            pass  # ❌ SILENT FAIL
```

**Проблемы:**
- `asyncio.create_task()` вызывается из watchdog thread, не из async context
- `run_until_complete()` блокирует watchdog thread
- `RuntimeError` молча игнорируется - emit НИКОГДА не происходит

```
    ↓

    ↓
RuntimeError caught silently ❌
    ↓
Socket.IO emit NEVER HAPPENS
    ↓
Frontend listener (empty anyway) never fires
    ↓
Tree doesn't update
```

```python
def _emit(self, event_name: str, data: Dict) -> None:
```
