## ROOT CAUSE: AsyncIO Event Loop Conflict

**Файл:** `src/scanners/file_watcher.py:382-404`

```python
def _emit(self, event_name: str, data: Dict) -> None:
    if loop and loop.is_running():
        loop.call_soon_threadsafe(
            lambda: asyncio.create_task(self.socketio.emit(...))  # ❌ BROKEN
        )
    else:
        asyncio.run(self.socketio.emit(...))  # ❌ BLOCKS THREAD
```

**Проблемы:**
1. `asyncio.create_task()` в lambda не работает правильно
2. `asyncio.run()` блокирует watchdog thread
3. Нет thread-safe механизма для Socket.IO emit
