### 2. Обнови `client/src-tauri/src/main.rs`

Добавь новые команды:

```rust
// MARKER_134_MYCELIUM_WINDOW: Mycelium Command Center floating window
#![cfg_attr(not(debug_assertions), windows_subsystem = "windows")]

use tauri::{Manager, WebviewUrl, WebviewWindowBuilder};

#[tauri::command]
async fn open_mycelium(app: tauri::AppHandle) -> Result<(), String> {
    if let Some(window) = app.get_webview_window("mycelium") {
        let _ = window.show();
        let _ = window.set_focus();
    } else {
        let _ = WebviewWindowBuilder::new(
            &app,
            "mycelium",
            WebviewUrl::App("/mycelium".into()),
        )
        .title("Mycelium Command Center")
        .inner_size(960.0, 680.0)
        .resizable(true)
        .always_on_top(false)
        .build()
        .map_err(|e| e.to_string())?;
    }
    Ok(())
}

#[tauri::command]
async fn close_mycelium(app: tauri::AppHandle) -> Result<(), String> {
    if let Some(window) = app.get_webview_window("mycelium") {
        let _ = window.hide();
    }
    Ok(())
}

fn main() {
    tauri::Builder::default()
        .invoke_handler(tauri::generate_handler![
            // ... твои существующие команды ...
            open_mycelium,
            close_mycelium,
        ])
        .run(tauri::generate_context!())
        .expect("error while running tauri application");
}
```

### 3. Создай standalone обёртку `client/src/MyceliumStandalone.tsx`

```tsx
/**
 * MyceliumStandalone — отдельное окно Command Center
 * Полностью независимое от основной Ветки
 */

import MyceliumCommandCenter from './components/dev/MyceliumCommandCenter';

export default function MyceliumStandalone() {
  return (
    <div style={{
      width: '100vw',
      height: '100vh',
      background: '#0a0a0a',
      overflow: 'hidden',
    }}>
      <MyceliumCommandCenter />
    </div>
  );
}
```

### 6. Как запустить и протестировать

1. Перезапусти Tauri:
   ```bash
   cd client
   yarn tauri dev
   ```

2. Нажми кнопку **MYCELIUM** — должно открыться отдельное окно.

3. Проверь:
   - Окно всегда можно перетаскивать
   - Можно сделать always-on-top (добавим позже)
   - Все графики и панели работают
   - WebSocket подключение к Mycelium работает
