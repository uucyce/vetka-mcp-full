# 🔬 GROK RESEARCH: Дерево не растёт!

## 🎯 ПРОБЛЕМА (переформулировано)

**Симптом:** Дерево показывается как ГОРИЗОНТАЛЬНЫЕ СТОЛБИКИ (дефолтный layout), а не как ПЕРЕВЁРНУТЫЙ DAG (корень внизу, ветки вверх).

**Что видим:** 2188 нод расположены в горизонтальной сетке — папки рядом друг с другом на одном уровне.

**Что нужно:** Ациклический направленный граф (DAG), перевёрнутый как настоящее дерево:
- Корень (root) ВНИЗУ
- Дети ВЫШЕ родителей
- Каждый уровень depth = выше по Y
- Fan-раскладка: дети расходятся веером от родителя

```
        [leaf]  [leaf]  [leaf]  [leaf]
           \      /        \      /
          [child]          [child]
               \            /
                 [parent]
                     |
                  [ROOT]  ← внизу
```

**История:** Вносили правки в fan_layout.py, на короткое время появилось 8 нод в правильном расположении, потом вернулось старое дефолтное дерево.
