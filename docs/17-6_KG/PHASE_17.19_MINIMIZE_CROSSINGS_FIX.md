## ROOT CAUSE

`minimize_crossings()` in `semantic_sugiyama.py` was **OVERWRITING all X positions** with a uniform grid, destroying the semantic layout from `distribute_by_similarity()`.

### Data Flow (BEFORE - BROKEN):
```
compute_adaptive_spread() → 156px           ✅ calculated
distribute_by_similarity() → [-42, +10, +80] ✅ semantic positions
minimize_crossings() → [-400, -267, -133]    ❌ ALL RESET TO UNIFORM GRID!
```

### The Problem:
```python
# OLD CODE - Created uniform spacing:
n = len(layer)
half_spread = x_spread / 2
for i, node_id in enumerate(layer):
    new_x = -half_spread + i * (x_spread / max(n - 1, 1))  # ← UNIFORM GRID!
    positions[node_id]['x'] = new_x  # ← OVERWRITES SEMANTIC X!
```

This created spacing = `x_spread / (n-1)` = `800 / 6` = **133px ALWAYS**

All similarity-based clustering was lost!

---

## THE FIX

**Sugiyama crossing minimization should:**
- ✅ REORDER nodes (sort by barycenter) - reduces edge crossings
- ❌ NOT RESET X positions to uniform grid - destroys semantic layout

**Solution:** Preserve the semantic X distances, only change which node gets which X value.

**NEW:**
```python
# PHASE 17.19 FIX: Preserve semantic X distances, only reorder
# Get current X values (preserves similarity-based spacing!)
current_xs = [positions[node_id]['x'] for node_id in layer if node_id in positions]

if len(current_xs) >= 2:
    # Sort X values and reassign to reordered nodes
    # This preserves the SPACING while changing the ORDER
    current_xs_sorted = sorted(current_xs)

    for i, node_id in enumerate(layer):
        if node_id in positions and i < len(current_xs_sorted):
            positions[node_id]['x'] = current_xs_sorted[i]
elif len(current_xs) == 1:
    # Single node - center it
    for node_id in layer:
        if node_id in positions:
            positions[node_id]['x'] = 0
```
