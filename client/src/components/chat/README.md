# vetka-chat-ui

Chat UI module with chat panel, sidebar history, input, and messaging components.

## Status
- Source of truth: monorepo danilagoleen/vetka
- Mirror sync: automated via subtree publish workflow
- Stability: experimental / fast-moving

## What Is Included
This repository contains the client/src/components/chat module extracted from VETKA.
It is intended for modular open-source collaboration and targeted contributions.

## Stack
- TypeScript + React
- Chat interaction components
- Sidebar/history/message UX

## Development
1. Fork this repository.
2. Create a branch: feature/short-name or fix/short-name.
3. Use Conventional Commits (for example, feat(api): add health endpoint).
4. Open a Pull Request with context and tests.
